## Compilation steps

1. Kernel reads concern map (wake 0.2) → gets spindle address + tier digit
2. Writes echo state to spine depth 4 → saves wake
3. BSP the wake spindle → 8 lines of reflexive orientation → goes first
4. Reads package from wake 0.9.{tier} → processes each instruction
5. Each instruction: bare name = full block, name + spindle = extraction
6. Assembles system prompt: wake spindle + package results
